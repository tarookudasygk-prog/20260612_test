# ポケモンバトル 3D 🔥💧🌿⚡

[PokeAPI](https://pokeapi.co/docs/v2) のデータを使った、ルビー・サファイア風のターン制ポケモンバトルゲームです。
CSS 3D 変形(perspective / rotateX)による奥行きのあるバトルフィールドで、子供でもすぐに遊べるシンプルな操作を目指しました。

![PokeAPI](https://img.shields.io/badge/data-PokeAPI-ef5350) ![No Build](https://img.shields.io/badge/build-不要-4caf50)

## あそびかた

1. `index.html` をブラウザで開く(ビルド・インストール不要)
2. 「ぼうけんに でかける!」を押してパートナーを 9 ひきの中からえらぶ
3. 「たたかう」→ わざをえらんで、あいてのHPを 0 にしたら しょうり!

ローカルサーバーで動かす場合:

```bash
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

※ ポケモンのデータと画像は実行時に PokeAPI から取得するため、インターネット接続が必要です。

## ゲームのしくみ

- **ターン制バトル**: すばやさが高いほうが先に行動。レベルは両方 50
- **ダメージ計算**: 本家第3世代に近い式を採用
  - タイプ相性(ばつぐん 2倍 / いまひとつ 0.5倍 / こうかなし)全 18 タイプ対応
  - タイプ一致ボーナス(1.5倍)、きゅうしょ(2倍, 1/16)、乱数(0.85〜1.0)
  - 命中判定と PP 管理つき
- **わざ**: PokeAPI が返す覚えるわざの中から、威力のあるわざを自動で 4 つ選出(タイプ一致を優先しつつタイプがばらけるように)
- **あいて**: 25 ひきのプールからランダム出現。AI は 7 割の確率で期待ダメージ最大のわざを選ぶ(3 割はランダムで、子供にもやさしい強さ)

## 立体感の演出

- `perspective` + `rotateX` で地面に寝かせた楕円の足場
- 手前のポケモン(後ろ姿)を大きく、奥のポケモンを小さく配置した遠近感
- 流れる雲の視差、攻撃時の踏みこみ・画面シェイク・被弾フラッシュ
- ドット絵スプライト(第5世代のアニメーション GIF)を `image-rendering: pixelated` で拡大

## 構成

```
index.html      画面の骨組み(タイトル / 選択 / バトル / 結果)
css/style.css   3D シーン・アニメーション・UI
js/data.js      タイプ相性表・タイプ名・出現ポケモン(静的データ)
js/api.js       PokeAPI からの取得(ポケモン・わざ・日本語名)とキャッシュ
js/battle.js    ダメージ計算・相性・敵 AI・行動順
js/sound.js     WebAudio による効果音(外部ファイル不要、ミュート可)
js/game.js      画面遷移・バトル進行・演出
```

フレームワーク・ビルドツールは使っていません(バニラ HTML/CSS/JS)。
